# CubeMart Email Service

![CubeMart](https://img.shields.io/badge/CubeMart-Email-0f766e)
![Python](https://img.shields.io/badge/Python-Service-3776AB)
![SMTP](https://img.shields.io/badge/SMTP-Notifications-1f2937)

Email microservice for sending CubeMart order confirmation notifications.

## Responsibilities

- Send order confirmation emails
- Render confirmation email templates
- Support SMTP-backed delivery
- Provide a dummy mode for local and test environments
